Vá para o terminal e dentro da pasta do seu projeto rode o banco de dados no modo servidor para um banco chamado loja-virtual:

	java -cp hsqldb.jar org.hsqldb.server.Server --database.0 file:loja-virtual --dbname.0 loja-virtual


Abra outro terminal e execute o manager para se conectar ao banco:

	java -cp hsqldb.jar  org.hsqldb.util.DatabaseManager
	
Execute no modo servidor do hsqldb.
 
Dentro do DatabaseManager execute a query para criar a tabela de produtos:

 	CREATE TABLE usuario (id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY, nome VARCHAR(255), senha VARCHAR(255)
