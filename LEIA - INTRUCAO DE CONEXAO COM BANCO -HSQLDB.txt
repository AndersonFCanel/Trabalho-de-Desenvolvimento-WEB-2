Vá para o terminal e dentro da pasta do seu projeto rode o banco de dados no modo servidor para um banco chamado loja-virtual:

	java -cp hsqldb.jar org.hsqldb.server.Server --database.0 file:dbweb2 --dbname.0 dbweb2

Abra outro terminal e execute o manager para se conectar ao banco:

	java -cp hsqldb.jar  org.hsqldb.util.DatabaseManager
	
Execute no modo SERVIDOR do hsqldb.
 
Dentro do DatabaseManager execute a query para criar a tabela de usuario:

 	CREATE TABLE usuario (idusuario INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
 	 nome VARCHAR(255), sobrenome VARCHAR(255), endereco VARCHAR(255), senha VARCHAR(255),
 	 login VARCHAR(255), dataNascimento VARCHAR(255), isAdm boolean NOT NULL	);
